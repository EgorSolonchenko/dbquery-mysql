Class QuerySplitter

Break down a mysql query statement to different parts, which can be altered and joined again. Supported types: SELECT, INSERT, REPLACE, UPDATE, DELETE, TRUNCATE.

SELECT ... UNION syntax is not supported. DELETE ... USING syntax is not supported. Invalid query statements might give unexpected results.

All methods of this class are static.
